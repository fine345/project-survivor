# 期末周 Dead Week

一款像素风生存射击手游，使用 Godot 4.6 开发。

## 游戏简介

学生角色在充满敌意的校园中生存，击败各种文具怪物和 BOSS。通过升级获取随机奖励强化角色，挑战三种难度模式。

## 主要特性

- **自动索敌射击** — 追踪子弹自动锁定最近敌人
- **虚拟摇杆** — 支持"任意位置"和"固定位置"两种模式
- **冲刺系统** — 松开摇杆触发无敌冲刺
- **升级奖励** — 每次升级随机提供 3 个强化选项
- **多种武器** — 旋转尺子、AI 激光
- **状态效果** — 冰冻、燃烧、护盾
- **3 个 BOSS** — 订书机、闹钟、学士帽，各有独特技能
- **3 种难度** — 正常、困难、挑战
- **防沉迷系统** — TapTap 实名认证

## 技术栈

- **引擎**：Godot 4.6（GDScript）
- **平台**：Android（竖屏 720×1440）
- **SDK**：TapTap 登录 & 防沉迷（FlowerTapTapSDK）

## 项目结构

```
scenes/game/     – 游戏场景（玩家、敌人、BOSS、子弹、经验球）
scenes/ui/       – UI 场景（主菜单、设置、成就、复盘）
scenes/effect/   – 状态效果（护盾、燃烧、冰冻）
scenes/weapon/   – 武器（尺子、计算器激光）
scripts/         – 对应的 GDScript 脚本
addons/          – 第三方插件（TapTap SDK、MCP 调试工具）
doc/             – 设计文档
assets/          – 精灵图、音频素材
```

## 运行

1. 使用 Godot 4.6 打开项目
2. 主场景为 `scenes/ui/login_screen.tscn`
3. PC 端自动跳过 TapTap 登录直接进入主菜单

## 导出 Android

1. 在 Godot 编辑器 **Project → Export** 中配置 Android 导出
2. 填写 Keystore 路径、用户名、密码
3. 点击 **Export Project** 生成 APK

## 难度说明

| 难度 | 敌人刷新间隔 | BOSS 血量倍率 | 解锁条件 |
|------|-------------|--------------|---------|
| 正常 | 1.0× | 1.0× | 默认 |
| 困难 | 0.67× | 1.5× | 通关正常模式 |
| 挑战 | 0.67× | 2.0× | 通关困难模式 |

## 控制

- **移动**：虚拟摇杆
- **攻击**：自动
- **冲刺**：松开摇杆
- **暂停**：右上角按钮
